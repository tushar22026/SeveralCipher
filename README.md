# SeveralCipher

## About This Project

SeveralCipher is a browser-based cryptography playground built with plain HTML, CSS, and JavaScript. It includes interactive tools for working with classic ciphers and adds simple cracking support for each cipher type.

### Features

- Monoalphabetic substitution cipher with a substitution map visualizer
- Playfair cipher with a 5x5 keyword grid
- Hill cipher using a 2x2 matrix
- Caesar cipher with shift visualization
- Vigenère cipher with key alignment feedback
- Live cracking helpers for: monoalphabetic, Playfair, Hill, Caesar, and Vigenère ciphers
- Client-side implementation with no backend required
- Responsive, neon-inspired UI for an engaging cryptography experience

## How to Use

1. Open `index.html` in a browser.
2. Select a cipher from the sidebar.
3. Enter your input text and the required key or parameters.
4. Click `Encrypt` or `Decrypt`.
5. Use the cracking button to run the built-in analysis or brute-force helpers.

## Notes

- The app runs entirely in the browser, so no installation is required beyond opening `index.html`.
- The cracking helpers are intended for learning and experimentation rather than guaranteed decryption of strong ciphers.
- Feel free to customize the UI, add more ciphers, or improve the cracking algorithms.
